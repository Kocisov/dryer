# drye

> Cloak create-react-app files with symlink.

## Installation

```bash
# install via yarn
$ yarn global add drye

# or npm
$ npm install -g drye
```

## Usage

```bash
# create new project
$ drye create <name>

# run cra command
$ drye run <command>

# install new package
$ drye add <package> [...other packages]
```
